# MaternIA
O MaternIA é um projeto desenvolvido como parte das atividades práticas da minha pós-graduação em Inteligência Artificial, com foco na aplicação de IA Generativa, Engenharia de Prompt e Gestão de Produtos.

O objetivo do projeto foi criar um assistente especialista capaz de responder exclusivamente com base em uma documentação oficial, atuando como uma fonte confiável de informações sobre um produto. Para isso, foram desenvolvidos dois artefatos fundamentais:

- PRD.pdf – Documento de especificação do produto, contendo seu escopo, funcionalidades e regras de negócio.
- Prompt.pdf – Documento responsável por definir a identidade, comportamento, limitações e diretrizes do assistente de IA, utilizando técnicas de Engenharia de Prompt para garantir respostas precisas, consistentes e livres de especulações.

O assistente foi projetado para atuar como um especialista no produto MaternIA, mantendo uma identidade fixa, respondendo apenas com informações presentes na documentação oficial e sendo transparente sempre que uma informação estiver fora do seu escopo.

## Objetivos do projeto
- Aplicar conceitos de Gestão de Produtos na criação de uma solução baseada em IA.
- Desenvolver uma documentação estruturada que sirva como base de conhecimento para um assistente especializado.
- Aplicar técnicas de Prompt Engineering, como Zero-Shot e Few-Shot Prompting.
- Construir um assistente confiável, transparente e alinhado às boas práticas de IA Generativa.
- Validar o comportamento do assistente utilizando o Google NotebookLM.

## Tecnologias e conceitos utilizados
- Inteligência Artificial Generativa
- Google NotebookLM
- Prompt Engineering
- Product Requirement Document (PRD)
- Engenharia de Contexto
- Zero-Shot Prompting
- Few-Shot Prompting
